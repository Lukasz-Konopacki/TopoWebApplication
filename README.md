# TopoWebApplication

Projekt aplikacji internetowej, służącej jako przewodnik wspinaczkowy. Aplikacja połączona jest z bazą danych, do której można dodawać opisy nowych dróg wspinaczkowych.

### Użyte Technologie

* Asp.net core Mvc
* Bootstrap
* Entity Framework
* Asp.Net Core Identity
